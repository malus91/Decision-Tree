Assignment 2-Implementation of Decsion tree using ID3 algorithm and random Attribute selection

Files Included

ID3Driver.java - This class contains the main function for triggering the flow of Dectree creation.  Calls different methods for building Decision tree using ID3 algorithm and random Attribute method.
  Methods:
  ID3Util.BuildDecTree()   -Build Tree using ID3
  RandomAttribute.BuildDecTree() -Build Tree using Random Attribute selection method
ID3Util.java -This class contains the dec tree build function and prune functions
DecTree.java- The class contains attributes and getter and setter methods for Decision tree
GetEntropy.java -the functions for calculation of entropy and information gain , populating datas to the files are present in this file.
RandomAttribute.java -Contains the method for building dec tree using random attribute selection

Compiation order and commands.
-> javac DecTree.java
-> javac GetEntropy.java
-> javac ID3Util.java -Xlint 
-> javac RandomAttribute.java -Xlint
-> javac ID3Driver.java
Execution command 
->The program accepts command line arguments if its given.Otherwise, the program will prompt for entering data. 
-> You can execute the program by using the following command

java ID3Driver 0 training_set.csv validation_Set.csv test_set.csv 0
